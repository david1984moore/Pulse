Basic Overview: 
  This app is the result of my personal struggle juggling finances in my head. Most budgeting apps are too cluttered, complicated, and don't give me insights on various scenarios based on the inout of the user. The goal is to be able to use this app to get a very simplistic view of my current financial situation, featuring an "income/bills" section, an interctive calendar, and a chatbot to handle complex user inputs. I want to be able to ask the chatbot "Can I spend $60 for dinner tonight?", the bot would reference the database, highlight bills on the calendar for the current week, and reply something like "Yes, you can spend $60. Account balance after dinner is $825. Be aware: electric bill ($300) is due on Monday. Accoutn balance after dinner tonight and electric bill, ($525)". And so I iterate.
